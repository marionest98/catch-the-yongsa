# 용사를 잡아라 (Catch the Yongsa)

## 소개
악마인 플레이어가 성물로 용사를 유인하고, 팬클럽으로 용사를 포위하는 턴제 전략 퍼즐 게임

## 장르
턴제 전략 퍼즐

## 개발 환경
- Unity / C#
- 1인 개발

## 게임 방법
1. 성물을 배치해서 용사를 유인
2. 팬클럽 + 벽으로 용사의 4방향을 막으면 승리
3. 용사 AI는 포위를 피해 영리하게 움직임
